## Introduction

The `FileForgeClient` is a powerful SDK for interacting with the FileForge API. It provides methods to generate and merge PDF documents from various web assets. This client handles communication with the FileForge service, making it easy to integrate into your projects.

## Installation

Before using the `FileForgeClient`, ensure you have the necessary dependencies installed. You can install the required packages using npm or yarn:

```sh
npm install fileforge
```

or

```sh
yarn add fileforge
```

## Usage

### Importing the Client

First, import the necessary modules from the `fileforge` package.

### Instantiating the Client

To instantiate the `FileForgeClient`, you need to provide your API key and optionally specify the environment.

```typescript
import { FileForgeClient } from 'fileforge';

const client = new FileForgeClient({
    apiKey: () => 'YOUR_API_KEY',
});
